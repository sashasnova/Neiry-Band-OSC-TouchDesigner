# Neiry Band → OSC → TouchDesigner

Этот код получает EEG-сигнал с гарнитуры **Neiry Band**, обрабатывает его в реальном времени и отправляет готовые данные в **TouchDesigner** (или любую другую программу) по протоколу OSC.

Написан для художников и перформеров — не для нейробиологов.  
Никакой науки. Просто сигнал, который можно трогать.

---

## Что выходит (OSC-каналы)

Все данные идут на `127.0.0.1:8000`

| адрес | тип | описание |
|---|---|---|
| `/neiry/arousal` | float 0–1 | **самый живой канал** — мгновенная активность: напряжение, концентрация, движение |
| `/neiry/ratio` | float 0–1 | расслабление: 1.0 = спокойно, 0.0 = напряжение |
| `/neiry/blink_env` | float 0–1 | моргание: 1.0 в момент моргания, затухает за ~350мс |
| `/neiry/blink` | int 1 | моментальный триггер моргания |
| `/neiry/alpha` | float 0–1 | альфа-ритм (8–13 Гц) — растёт когда закрыты глаза |
| `/neiry/beta` | float 0–1 | бета-ритм (14–30 Гц) — активное мышление, тревога |
| `/neiry/theta` | float 0–1 | тета-ритм (4–7 Гц) — медитация, дрёма |
| `/neiry/hb` | int N | счётчик heartbeat (для проверки что сигнал идёт) |

> Сигналы стабилизируются примерно за 8–10 секунд после надевания гарнитуры.  
> Проверка ratio: закрой глаза на 5 секунд — значение должно ползти вверх.

## Что нужно перед установкой

1. **Гарнитура Neiry Band** + Bluetooth на компьютере
2. **Capsule SDK** — скачать у Neiry (версия 1.3.0 или выше)  
   Папка после установки выглядит так:
   ```
   D:\Capsule_and_API_1_3_0_22_03_2024\
     capsule-public-v1.3.0\
       CapsuleAPI\
         Win\
           Source\      ← сюда кладём файлы
           build\
   ```
3. **Visual Studio** (Community бесплатная) с поддержкой C++
4. **TouchDesigner** или любая другая программа с поддержкой OSC

## Установка — шаг за шагом

### Шаг 1 — скопировать файлы

Скачай три файла из этого репозитория и положи в папку:
```
CapsuleAPI\Win\Source\
```

Файлы:
- `main.cpp` (это RawSignalExample.cpp — переименуй если нужно)
- `EEGBands.hpp`
- `OscSender.hpp`

### Шаг 2 — открыть проект в Visual Studio

Открой файл `.sln` или `CMakeLists.txt` из папки:
```
CapsuleAPI\Win\build\
```

### Шаг 3 — собрать

В Visual Studio: меню **Build → Build Solution** (или `Ctrl+Shift+B`)

Готовый `.exe` появится в:
```
CapsuleAPI\Win\build\build\Release\
```

### Шаг 4 — запустить

Запусти `CapsuleRawSignalExample.exe`  
Программа спросит: **Write raw data to CSV? (y/n)** — жми `n` если не нужно.

После этого данные пойдут по OSC на порт 8000.

---

## TouchDesigner — как принять сигнал

1. Добавь ноду **OSC In CHOP**
2. Поставь: Network Address `127.0.0.1`, Port `8000`
3. В параметрах выбери **Scope: /neiry/***

Каналы появятся автоматически когда придут первые данные.
